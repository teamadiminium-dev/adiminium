The Privacy Guide to EXIF Scrubbing

The Adiminium Metadata Remover is your digital shredder for invisible data. While the visual part of your image (the pixels) is what you want to share, the invisible part (the metadata) often shares more than you intend. This tool creates a clean COPY of your image with zero metadata attached.

Security Risk: Cyberstalkers can use the GPS data embedded in a simple photo of a pet or a view from a window to locate a person's exact home address. Always scrub photos before posting to public forums or classifieds (like Craigslist).

What Gets Removed?

  • GPS Location: Latitude, Longitude, Altitude, Direction of travel.
  • Camera Details: Serial Number, Owner Name (if set in camera).
  • Settings: Shutter count, Exposure bias, Flash mode.
  • Software Info: "Edited with Photoshop", etc.
  • Thumbnails: Many cameras embed a tiny low-res preview inside the main file. Sometimes this thumbnail shows the original unedited version of a photo you later cropped! We remove this too.

How It Works

Unlike simple "text editors" that try to cut out text strings, our tool parses the image pixels and re-draws them onto a fresh HTML5 Canvas. It then exports a brand new image file. This ensures that only the visual data is carried over. It is physically impossible for the old metadata to survive this process.

Step-by-Step Guide

  1. Drag & Drop: Place your sensitive image in the box.
  2. Auto-Clean: The browser immediately re-processes the image.
  3. Verify: We show you the file size difference. (Metadata can take up 10-50KB!).
  4. Download: Save the "clean_" version.

Frequently Asked Questions

Does quality degrade?

Technically, yes, because we are re-compressing a JPEG. However, we use a high-quality setting (92/100) which generally creates a visually identical result.

Can I scrub multiple files?

Currently this tool processes one image at a time to ensure maximum security and browser stability.
